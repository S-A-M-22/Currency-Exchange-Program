# Currency Exchange Program

This github repository contains **Currency Exchange Program**. 

# Run the Program
To run this program it is recommended that you run the command: ```gradle build``` and then ```gradle run``` in the root directory of the project.

# Login Process
You will have to option to either login as a normal user or as an admin.

To login as a Normal User:
Enter **'user'** or **'normal user'** in the username textfield and write **'password'** in the password textfield then press the ```Login``` button.

To login as an Admin:
Enter **'admin'** in the username textfield and write **'superpassword'** in the password textfield then press the ```Login``` button.

Now you will see all the options that you can access on the window.

# Run User-specific Tasks

**For Normal Users:**
The screen will display the following commands you can carry out:
- ```Convert Currency```
- ```View Popular Currencies```
- ```View History```

You can now select any option you like and it will prompt you to a different screen where you have to enter the values as required by the input fields and click the ```Submit``` button at the end to complete the specific task.

**For Admin:**
The screen will display the following commands you can carry out:
- ```Add New Currency```
- ```Update Exchange Rates```
- ```Convert Currencies```
- ```View Popular Currencies```
- ```View History```

You can now select any option you like and it will prompt you to a different screen where you have to enter the values as required by the input fields and click the ```Submit``` button at the end to complete the specific task.

# Testing

You can also run tests on the program by running the command: ```gradle clean test```
